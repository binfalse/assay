# ASSAY

Like an essay, but by an ass. Inspired by an [Regular Old Bean](https://octodon.social/@faho/103599063487089173).

## Dependencies

    pip3 install playsound

## Run

    git clone https://github.com/binfalse/assay
    cd assay
    python3 assay.py "your essay"
